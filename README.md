# Camera Pose Calibrator

* #### 💡 개요
  * Setup 상황에서 카메라가 정(正)자세로 설치되었는지 여부 검증 및 교정
  * 이미지 사이즈, FOV, Working Distance 을(를) 알고있을 때 카메라 행렬(Camera Matrix) 계산
  * 카메라의 Roll, Pan, tilt 각도를 계산하여 표시
   
* #### 🌍 환경 & 호환성  
  * OS:	MS Windows 10 ~
  * Language : C++ 11 ~
  * Opencv version : 4.6.0 ~
